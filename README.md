# Network Compliance & Regulatory Frameworks

## 📌 Overview
This project demonstrates how to implement and check network systems against major government and industry compliance standards such as ISO 27001, NIST, and SOC 2.

The repo contains practical simulations, policy templates, and mapping diagrams showing how network configurations can comply with regulatory requirements.

---

## 🛠️ Compliance Frameworks Covered
- **ISO 27001:** Information Security Management System (ISMS)
- **NIST SP 800-53:** Security controls for federal systems
- **SOC 2:** Trust service principles for service organizations
- **HIPAA (optional):** Health data protection for network systems
- **PCI DSS (optional):** Payment data security

---

## 📂 Project Files
- `compliance_checker.py` → Python script simulating policy checks
- `policies/` → Example network policies mapped to compliance standards
- `diagrams/compliance_mapping.png` → Diagram mapping network layers to compliance frameworks
